## 🚀 Features

* **Modern Layout:** Clean split-column layout designed with custom CSS variables.
* **Separation of Concerns:** Styled completely with an external stylesheet (`style.css`), free of comments.
* **Photo Ready:** Features a rounded profile image placeholder container built into the layout.
* **Print Optimized:** Automatically adjusts background styles via `@media print` for saving cleanly as a PDF.
